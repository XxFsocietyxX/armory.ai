# ARMORY.AI Portfolio

This is a premium 3D portfolio website showcasing your weapon and armor assets.

## Tech Stack
- **React** + **Vite**
- **Three.js** + **React Three Fiber** (3D Rendering)
- **TailwindCSS v4** (Styling)
- **Framer Motion** (Animations)

## How to Run
1. Open this folder in your terminal:
   ```bash
   cd agency/portfolio
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:5173 in your browser.

## Customization
- **Models**: Add new `.glb` files to `public/models/`.
- **Data**: Update `src/data.js` to add/remove items or adjust their scale/position.
- **Styling**: Edit `src/index.css` to change the color palette or fonts.
